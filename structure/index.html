# HTML 语义化

# H1~H6 标签

<h1>~<h6> ，作为标题使用，并且依据重要性递减，<h1> 是最高的等级。

<h1>包含核心关键词，它可以是网站的 logo、网站导航的第一个链接。

一个页面中有且仅有一个<h1>

# 自带样式标签

在页面中尽可能不使用以下标签

标签 作用 替代
<b> 文本加粗 font-weight: 600、<strong>
<u> 文本下划线 text-decoration: underline
<i> 文本倾斜 font-style: italic、<em>

# p 标签

<p>作为段落标记，尽量不使用<br>来使其换行。

段落间隔使用 CSS 控制。

# table 标签

table等其附属标签请用于制作表格，不做布局使用。

# 自闭合标签不需要斜杠

<br>、input等自闭合标签不需要写斜杠 <br /> 不需要这样写。
